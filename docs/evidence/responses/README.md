# Responses / Screenshots

Place screenshots and saved responses from Twitter, Reddit, and emails here.

Filename pattern (recommended):
`YYYYMMDD_HHMMSS_platform_shortdesc.png`
example: `20251217_104532_twitter_reply_janedoe.png`

Usage:

- Save full-thread screenshots, important replies, and Reddit comment screenshots here.
- Add an entry to `docs/evidence/response_log.md` referencing the filename and a short summary.

Privacy:

- Avoid storing private messages verbatim unless you have permission. Instead, record date, channel, and a one-line summary.
