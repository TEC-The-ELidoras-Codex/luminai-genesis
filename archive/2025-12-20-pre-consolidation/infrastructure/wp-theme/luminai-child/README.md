LuminAI Child Theme

A small starter child theme for the LuminAI WordPress site. This provides a minimal style and a page template for publications.

Installation steps:

1. Zip this folder contents and upload it to your WP site under Appearance -> Themes -> Add New -> Upload Theme.
2. Activate the theme or switch to a child theme of an existing parent theme.
3. Place `page-templates/publications.php` into the theme's folder to make the publications template available when creating/editing a page.

The child theme includes a `Publication Sidebar` widget area for publication cards.
