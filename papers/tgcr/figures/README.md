Figures for TGCR paper

Expected filenames (place in this `figures/` folder):

- `tgcr_diagram.png` — schematic of R, W, R'
- `ai_results.png` — scatter/line showing observed vs predicted R'
- `quantum_setup.png` — diagram of EEG + quantum experiment setup

Image guidelines

- Preferred formats: PNG, JPG, PDF (PDF for vector images)
- Recommended resolution for raster images: 300 dpi
- Recommended color space: RGB for arXiv, convert to grayscale if necessary for print

If you don't have images yet, the LaTeX source will render a framed placeholder box indicating the missing filename so compilation won't fail.
