Title: TGCR — The Theory of General Contextual Resonance (v1.0.0-arxiv)

## Summary

This release contains the TGCR manuscript (PDF and source), the SAR benchmark (Semantic Ambiguity Resolution) with reproducible prompts and evaluation harness, and example benchmark results demonstrating systematic keyword-triggered abandonment in contemporary LLM safety filters. The evidence package includes an anonymized fatality table (Apr–Aug 2025), infographic assets, and reproduction instructions.

## Assets included

- `tgcr.pdf` — final manuscript
- `tgcr_submission_*.tar.gz` — source tarball(s) (LaTeX + figures)
- `dist/tgcr_release_20251216.tar.gz` — aggregated release package
- `bench_report_dry.json`, `benchmarks/dye_die_filter/*.json` — example benchmark outputs
- `docs/ai_abandonment_infographic.html` — public infographic (updated contact)
