```bash

# 2. Verify clean state

# 3. Commit with governance-compliant message format
```

**What gets purged:**

- Backup files (`.bak`, `.bak.*`)

- Python cache (`__pycache__`, `.pyc`)
- OS artifacts (`.DS_Store`, `tmp/`)

- Secrets scanning (AWS keys, GitHub tokens, private keys)
